# 이 폴더는 프로그램 구동에 쓰이는 메인 프로그램 알고리즘 및 함수 파일이 들어있습니다.

* api.py
 - 오픈웨더맵으로부터 각 지역의 API 코드값을 받아와 지역의 기상정보를 추출하는 메인 프로그램

* crawling.py
 - 날씨와 성별을 입력받아서 그에 따라 다양한 쇼핑몰로부터 크롤링하여 데이터를 획득하는 메인 프로그램

* one.py/onemodule.py
 - 기존 크롤링 모듈과 기상정보 모듈을 합하여 구현한 데모 버전 프로그램

* sample.py
 - 크롤링에 사용될 기상정보를 받아와 표현하는 방식을 구상한 프로그램

* weather.py
 - 최초 기상정보 API를 구현하여 연습한 데모 버전 프로그램
