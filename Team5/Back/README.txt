# 이 폴더는 웹 구동을 위한 Back-end 디렉토리로 프로그램의 메인 디렉토리 입니다.

* app 디렉토리 안으로 python flask를 통한 백엔드 구현
 
 app
   ㄴ module
         ㄴ link
   ㄴ static
         ㄴ css
         ㄴ images
         ㄴ js
   ㄴ templates

* module 디렉토리는 프로그램에 사용되는 모듈을 저장한 공간

* link 디렉토리는 크롤링을 통하여 획득한 데이터를 저장하기 위한 폴더

* static 디렉토리는 bootstrap과의 연동을 위한 bootstrap 데이터를 저장

* images 디렉토리는 웹에 표현되는 이미지 파일을 저장한 공간

* templates 디렉토리는 html 파일 저장소

* run.py 를 통하여 서버 구동 가능, 내부적으로 본인의 IP를 설정하여 사용 가능
